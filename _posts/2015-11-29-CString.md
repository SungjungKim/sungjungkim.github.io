## strcpy
문자열을 복사한다.

| 헤더     | string.h                                                |
| ------- | ------------------------------------------------------- |
| 형태     | char *strcpy(char *dest, const chat *src)               |
| ------- | ------------------------------------------------------- |
| 인수     | char *dst : 복사할 위치                                    |
|         | char *src : 원문 문자열                                    |
| ------- | ------------------------------------------------------- |
| 반환     | 복사한 문자열을 반환                                         |

    
    #include <stdio.h>
    #include <string.h>
    
    int main(void) {
        char buf[50];
        char *ptr;
        
        ptr = strcpy(buf, "sungjung.github.io");
        printf("%s\n", buf);
        printf("%s\n", ptr);
        
        return 0;
    }
